# GuessingApp #

## UC1 - Game Initialization ##
    - Initialize guessing game with required configurations

## UC2 - User Guess Submission ##
    - Allow the player to submit a guess and recieve immediate feedback.

## UC3 - Hint Generation ##
    - Provide helpful hints to guide the player toward the correct answer.

## UC4 - Error Handling and Validation ##
    - Validates user inputs throughout the game.

## UC5 - Storage Service ##
    - Store and retrieve game results for future reference

## UC6 - Game Restart and Exit##
    - Allow the player to restart or exit the game safely.
